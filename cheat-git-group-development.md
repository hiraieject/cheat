
# cheat sheet [group development]

##
    $ git clone ［リポジトリパス］
    リモートリポジトリをローカルにクローンする

    $ cd <リポジトリディレクトリ名>
    $ git branch -a
    ブランチの一覧を確認

    $ git branch -a
    * master
      remotes/origin/HEAD -> origin/master
      remotes/origin/master
    最初はこうなっているはず
