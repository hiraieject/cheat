
# cheat sheet [initialize git]

## gitの設定
   すべてのローカルリポジトリ用の、ユーザー情報設定方法

        $ git config --global user.name "[name]"
        コミット操作に付加されるあなたの名前を設定します

        $ git config --global user.email "[email address]"
        コミット操作に付加されるあなたのメールアドレスを設定します

## gitのインストール

        $ sudo apt update
        $ sudo apt install git
        ubuntu系
